Introduction

This application is developed as part of a technical task for Danske Bank. It provides functionality for managing and retrieving municipal tax rates based on different time periods. The system is implemented in Java using Spring Boot.

Technical Task Description

The application manages taxes applied in different municipalities. It supports scheduling tax records with specific validity periods and enables users to query the applicable tax rate for a municipality on a given date.

Example Scenario

Municipality Copenhagen has the following scheduled taxes:

  • Yearly tax: 0.2 — from Jan 1, 2025 to Dec 31, 2025
  • Monthly tax: 0.4 — from May 1, 2025 to May 31, 2025
  • No weekly taxes scheduled
  • Daily taxes: 0.1 on Jan 1, 2025 and Dec 25, 2025
